# GPT補足報告: Quantum Lattice / Droplet / History Terrain

この補足は、会話内で進めた `v1.1 → v1.5 → v2 → droplet/history` の流れを、既存の `REPORT_2026-07-07.md` に追加する形で整理したものです。

## 0. 目的

この一連の実験は、量子格子を使って「より組織化した物質的挙動」を作れるかを探索したものです。

ただし、途中で重要な規律が確立しました。

- 生命語彙を使いすぎない。
- 古典対照に食われる現象を量子証拠と呼ばない。
- 予想と違う結果を隠さない。
- UIではなく数学コアを優先する。

## 1. v1.1: 4量子ビット反応体

構成:

```text
S  = 感覚
M1 = 浅い記憶
M2 = 深部記憶
R  = 応答
```

分かったこと:

- M2へ書くには `M1→M2 = CRY/CRX` が必要。
- `CRZ` は target が `|0>` に近いと書けない。
- drift により tempo island が出た。
- ただし「M2半回転則」は κスイープで反証済み。
- 正しい読みは、単純共鳴ではなく多島構造。
- `dephase` で島が消える点が重要。これは位相コヒーレンス依存の witness。

## 2. v1.2: Adaptive Coupling

相関に応じて `adaptiveG` を育てる層を追加。

分かったこと:

- high learning / low forgetting / low threshold では全面ゲル化する。
- 休み/剪定を入れると選択的リンク形成に近づく。
- しかし現象の主役は古典的 adaptive layer へ移動していた。

## 3. v1.5: Spatial Metastable Matter

3×3セルで核生成・伝播・ドメイン境界を見た。

結果:

- 中央刺激で局所核が立つ。
- `spatial` を上げると伝播する。
- 強すぎると全体ゲル化。
- 左右二核で界面っぽい中立帯が出る。

しかし、量子ビット削除テストで主要現象は古典CAでも再現された。

結論:

```text
v1.5は量子反応体としては古典に食われた。
```

## 4. v2: EXCH 量子格子と negativity

設計を完全量子格子へ戻した。

主ゲート:

```text

```

重要な切り分け:

## 5. v3: EXCH + ZZ

`ZZ` を入れると束縛ペア/クラスターが出るかを調査した。

結果:

- 既に隣接した2励起/3励起はZZで保たれる。
- 離れた励起はZZだけでは合体しにくい。
- 強すぎるZZは液滴ではなく凍結を作る。

結論:

```text
ZZは塊を保つ。
しかし合体を不可逆化するには散逸が要る。
```

## 6. v4: Surface loss と履歴地形

通過履歴 `h_i` を地形として持たせ、次の散逸率を変えた。

2種類:

```text
attract:
  履歴がある場所のlossを弱める

repel:
  履歴がある場所のlossを強める
```

予想は「attractで道を辿る」だったが、結果は逆寄りだった。

結果:

- attract は古い道への overlap を上げるが、液滴をほどいた。
- repel は古い道を避けるが、液滴を締めた。
- 履歴追従 ≠ 組織化。
- 履歴排除は尾を切り落とし、コンパクトな頭を残す。

## 7. Claude/GPT 統合アブレーション

repel 凝集が量子か古典かを確認した。

結果:

- repelで締まる現象自体は古典確率モデルでも出る。
- したがって、repel凝集だけを量子性の証拠にしてはいけない。
- ただし、履歴地形上での輸送・広がり・経路選択は初期位相に依存した。

切り分け:

```text
凝集 = 古典寄り
輸送/経路 = 量子位相寄り
```

## 8. Edge memory

履歴を bulk と edge に分けた。

```text
bulk memory:
  液滴内部が通った跡

edge memory:
  液滴の端/境界が通った跡
```

結果:

- bulk memory は弱い。
- edge memory のほうが液滴の進行・締まりに効く。
- ただし現時点では「表面張力」そのものではなく、通過済み境界に残った尾を削る履歴的散逸地形として効いている。
