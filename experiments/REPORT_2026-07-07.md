## 0. 記録修正

以前の「M2半回転則」は反証済み。

```text
κ=0.71では見かけ上一致したが、κスイープでは一般則として不成立。
現在の正しい読みは、単純共鳴ではなく多島構造。
```

この修正は以後の解釈の前提にする。

## 1. v1.1: 4量子ビット反応体

構成:

```text
S  = 感覚
M1 = 浅い記憶
M2 = 深部記憶
R  = 応答
```

主結果:

- `M1→M2` は `CRY/CRX` でないと深部へ書き込みにくい。
- `CRZ` は target が `|0>` に近いと沈黙しやすい。
- drift を入れると、`S S S D` のような系列にテンポ島が出た。
- ただし単純な共鳴則ではなく、多島構造だった。
- dephase でテンポ島が消えた。これは、系の面白さが位相コヒーレンスに載っていることを示す最初の強い手がかり。

## 2. v1.2: adaptive coupling

相関を古典的な `adaptiveG` として保存する層を追加した。

結果:

```text
高学習・低忘却・低しきい値 → 全面ゲル化
低学習・高忘却・高しきい値 → 選択的リンク形成
```

ユーザー保存標本では全リンクが上限付近まで太り、全面ゲル化と判断した。

重要な反省:

```text
面白い現象が量子層ではなく古典適応層に移動し始めた。
```

境界 `B`、餌/エネルギー `E`、触媒核 `M2` を追加した。

```text

境界を人工的に置いても、生命に近づいたわけではない。
```

## 5. v1.5: spatial metastable matter

3×3セル格子を導入。局所3量子ビット回路 + 古典的な空間凝集場。

結果:

- 中央刺激で局所核生成。

判定:

## 6. v2: quantum lattice / EXCH / negativity

FABLEの指摘により、逆位相ドメイン壁だけでは古典波動に食われることを確認。

設計:

```text
9量子ビット1D chain
EXCH(θ)=exp(-iθ(XX+YY)/2)
古典波動 ψ 対照
古典CA 対照
```

結果:

### A. 逆位相二核

```text
(|q1> - |q7>) / √2
```

中央 q4 は節になるが、古典波動と完全一致。

判定:

```text
位相壁はCAキラーだが、量子証拠ではない。
```

### B. 二粒子衝突

```text
q1=|1>, q7=|1>
```

独立古典波動とはズレた。

例:

```text
tick 20:
  quantum same-side = 0.412
  independent wave = 0.267

tick 27:
  quantum center-involving = 0.753
  independent wave = 0.475
```

ただし反対称Slater determinantとは一致。主証拠にはしない。

### C. negativity filament

隣接bond negativity:

```text
tick 2   maxN = 0.230
tick 3   maxN = 0.271
tick 4   maxN = 0.254
tick 12  maxN = 0.203
```

dephaseで消える:

```text
eta=1.0  maxN=0.2705
eta=0.8  maxN=0.1519
eta=0.5  maxN=0.0363
eta=0.2  maxN=0.0011
eta=0.0  maxN=0
```

判定:

```text
v2の本命は逆位相壁ではなく、negativity filament。
```

## 11. 古典/量子アブレーションと edge-memory

Claudeの賭け:

```text
repelで締まる現象は大部分が古典的自己回避で説明でき、位相寄与は小さい。
```

結果:

```text
mode       Quantum P_cluster   Classical P_cluster
none          0.118                0.464
attract       0.073                0.350
repel         0.131                0.522
bulk_repel    0.119                0.459
edge_repel    0.129                0.527
```

凝集そのものは古典自己回避でも出る。

ただし位相を変えると輸送は変わった。

```text
initial phase     P_cluster   mean position   IPR
k = +π/2            0.131        5.52          72.6
k = 0               0.026        7.25         193.6
k = -π/2            0.031        7.46         175.2
random phase avg    0.110        5.79          99.6
```

切り分け:

```text
repel凝集 = 古典的成分が大きい
repel地形上の量子液滴輸送 = 位相依存が残る
```

### edge-memory

履歴を bulk / edge に分離。

```text
bulk memory:
  液滴内部が通った跡

edge memory:
  液滴の端/境界が通った跡
```

自分で履歴を書きながら走る場合:

```text
mode          P_cluster   mean position   IPR
none            0.109        7.19        116.5
bulk_repel      0.111        7.36        116.6
edge_repel      0.133        8.13        103.2
repel both      0.136        8.30        100.3

bulk_attract    0.094        7.02        119.0
edge_attract    0.056        7.80        147.5
```

判定:

```text
bulk_repelはほぼ効かない。
edge_repelはP_clusterを上げ、IPRを下げる。
```

ただし現時点では、edge-memoryを表面張力そのものとは呼ばない。

より正確には:

```text
edge-memory repel = 尾の切り落とし機構
```

```text
自己回避的凝集は古典寄り。
しかし、edge履歴地形上での輸送・広がり・経路選択には量子位相が効く。
```

次の本命:

```text
phase-routed self-avoiding quantum droplet
```

問い:

```text
edge履歴地形を作る
↓
同じ地形に位相だけ違う液滴を入れる
↓
進路・分裂・再凝集・凝集率が変わるか
↓
古典確率対照と比較する
```

これが、現在もっとも未知に近い方向。
